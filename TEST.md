## 🚀 كيفية الاختبار:

### اختبار 1: الأيقونات والاسم
1. افتح المتصفح (Chrome/Edge)
2. اذهب إلى الرابط
3. **للموبايل:**
   - Safari (iOS): Share → Add to Home Screen
   - Chrome (Android): Menu → Install App
4. **للكمبيوتر:**
   - Chrome: أيقونة التثبيت في شريط العنوان
5. **تحقق من:**
   - ✓ الأيقونة تظهر بشكل صحيح
   - ✓ الاسم "ونّسني" يظهر
   - ✓ التطبيق يفتح full screen

### اختبار 2: التحديثات التلقائية
1. افتح DevTools (F12)
2. اذهب لـ Console
3. ابحث عن: `✅ Service Worker registered`
4. **لاختبار التحديث:**
   - غيّر أي شيء في الكود
   - انتظر 60 ثانية أو اضغط 🔄
   - يجب أن يظهر إشعار التحديث
5. **تحقق من:**
   - ✓ إشعار أرجواني يظهر في الأسفل
   - ✓ زر "تحديث الآن" يعمل
   - ✓ التطبيق يُحدّث تلقائياً

### اختبار 3: الرابط
1. افتح: `your-domain.com/wansni`
2. أنشئ قعدة كأدمن
3. **تحقق من QR Code:**
   - امسح الكود بالموبايل
   - يجب أن يفتح: `your-domain.com/wansni?room=1234`
4. **تحقق من:**
   - ✓ الرابط يحتوي على `/wansni`
   - ✓ QR Code يعمل
   - ✓ الانضمام يعمل بشكل صحيح

## 🔍 التحقق من Service Worker:

### في Chrome DevTools:
1. اضغط F12
2. اذهب لـ Application
3. اذهب لـ Service Workers
4. **يجب أن ترى:**
   - ✓ Status: activated and running
   - ✓ Source: service-worker.js
   - ✓ Scope: /

### في Cache Storage:
1. اذهب لـ Application → Storage → Cache Storage
2. **يجب أن ترى:**
   - ✓ wansni-cache-[timestamp]
   - ✓ الملفات المخزنة (/, /wansni, icons, manifest)

## 🐛 استكشاف الأخطاء:

### الأيقونات لا تظهر؟
```bash
# تحقق من وجود الملفات
ls -lh public/icon-*.png

# تحقق من manifest
cat public/manifest.json
```

### Service Worker لا يعمل؟
1. تأكد أنك على HTTPS (أو localhost)
2. افتح Console وابحث عن أخطاء
3. امسح الـ cache: DevTools → Application → Clear storage

### التحديثات لا تظهر؟
1. انتظر 60 ثانية
2. اضغط زر 🔄
3. تحقق من Console
4. امسح Cache وأعد التحميل
