# 🎉 التحديثات الجديدة - ونّسني

## ✨ ما الجديد؟

### 1. 📱 Progressive Web App (PWA)
اللعبة أصبحت تطبيق ويب تقدمي كامل!

**المميزات:**
- ✅ يمكن تثبيتها على الموبايل كتطبيق حقيقي
- ✅ تعمل أوفلاين (ما عدا التزامن المباشر)
- ✅ تحديثات تلقائية لجميع المستخدمين
- ✅ أيقونات مخصصة (192x192 و 512x512)
- ✅ شاشة بداية سريعة

### 2. 🔄 نظام التحديثات التلقائية

**كيف يعمل؟**
1. عند نشر نسخة جديدة، يكتشفها Service Worker تلقائياً
2. يظهر إشعار جميل للمستخدمين في أسفل الشاشة
3. المستخدم يضغط "تحديث الآن"
4. التطبيق يُحدّث فوراً بدون إعادة فتح

**الفحص التلقائي:**
- كل 60 ثانية
- عند فتح التطبيق
- عند الضغط على زر 🔄

### 3. 🔗 روابط محسّنة

الرابط الآن يحتوي على اسم اللعبة:
```
https://your-domain.com/wansni
```

**رابط الانضمام:**
```
https://your-domain.com/wansni?room=1234
```

### 4. 🎨 إشعار التحديث المحسّن

إشعار جميل يظهر عند توفر تحديث:
- تصميم أرجواني أنيق
- رسوم متحركة سلسة
- زرين: "تحديث الآن" و "لاحقاً"
- يختفي تلقائياً بعد التحديث

### 5. 🖼️ الأيقونات المخصصة

تم إضافة أيقونات اللعبة:
- `icon-192.png` - للشاشات الصغيرة
- `icon-512.png` - للشاشات الكبيرة
- تعمل كـ maskable icons (تتكيف مع شكل الجهاز)

## 📂 الملفات الجديدة

```
public/
├── icon-192.png              ← أيقونة 192x192
├── icon-512.png              ← أيقونة 512x512

src/

UPDATES.md                    ← هذا الملف
```

## 🚀 كيفية الاستخدام

### للمطورين:

1. **التطوير المحلي:**
```bash
pnpm install
pnpm dev
```

2. **البناء:**
```bash
pnpm build
```

3. **النشر:**
انظر `DEPLOY.md` للتفاصيل الكاملة

### للمستخدمين:

1. **تثبيت التطبيق:**
   - افتح الرابط في المتصفح
   - اضغط على "Install App" أو "Add to Home Screen"

2. **التحديث:**
   - سيظهر إشعار تلقائياً عند توفر تحديث
   - اضغط "تحديث الآن"
   - أو اضغط زر 🔄 في الزاوية

3. **اللعب أوفلاين:**
   - بعد أول زيارة، التطبيق يعمل بدون إنترنت
   - التزامن المباشر يحتاج إنترنت

## 🔧 التقنيات المستخدمة

- **Vite PWA Plugin** - لإدارة Service Worker
- **Workbox** - للـ caching الذكي
- **Web App Manifest** - للتثبيت والأيقونات
- **Service Worker** - للعمل أوفلاين والتحديثات

## 📊 الأداء

### Cache Strategy:

**CacheFirst (Firebase Storage):**
- ملفات الوسائط
- الصور والأيقونات
- مدة: 30 يوم

**NetworkFirst (Firebase DB):**
- بيانات Realtime Database
- مهلة: 10 ثواني
- تراجع للـ cache عند فشل الاتصال
