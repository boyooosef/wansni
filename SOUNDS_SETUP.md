# 🔊 إعداد الأصوات - WanSni

## ✅ تم إضافة نظام الأصوات!

تم تفعيل نظام الأصوات بالكامل في اللعبة. الآن كل ما تحتاجه هو إضافة ملفات الأصوات.

## 📁 الملفات المطلوبة:

ضع ملفات MP3 في مجلد `public/sounds/`:

```
public/sounds/
├── vote.mp3         ← صوت عند التصويت (نجح/فشل)
├── spin.mp3         ← صوت دوران العجلة (اختيار اللاعب)
├── tick.mp3         ← صوت آخر 5 ثواني من المؤقت
├── correct.mp3      ← صوت الإجابة الصحيحة (النجاح)
├── wrong.mp3        ← صوت الإجابة الخاطئة (الفشل)
└── winner.mp3       ← صوت الفوز بالقعدة
```

## 🎵 متى تُشغّل الأصوات؟

### 1. `vote.mp3` - صوت التصويت
- عند ضغط ✅ نجح
- عند ضغط ❌ فشل
- **الحجم:** 70% (0.7)

### 2. `spin.mp3` - صوت دوران العجلة
- عند ضغط "اختيار اللاعب القادم"
- أثناء دوران العجلة (35 دورة)
- **الحجم:** 40% (0.4)

### 3. `tick.mp3` - صوت المؤقت
- آخر 5 ثواني من وقت الإجابة
- يُشغّل كل ثانية: 5...4...3...2...1
- **الحجم:** 30% (0.3)

### 4. `correct.mp3` - صوت النجاح
- عند اعتماد التصويت والنتيجة نجاح
- **الحجم:** 80% (0.8)

### 5. `wrong.mp3` - صوت الفشل
- عند اعتماد التصويت والنتيجة فشل
- **الحجم:** 80% (0.8)

### 6. `winner.mp3` - صوت الفوز
- عند وصول لاعب للنقاط المطلوبة
- **الحجم:** 100% (1.0) - أعلى صوت!

## 🔧 الكود المُضاف:

### ملفات جديدة:
```
✅ src/app/hooks/useSound.ts    ← نظام الأصوات
```

### ملفات محدثة:
```
✅ AdminScreen.tsx    → spin, vote, correct, wrong, winner
✅ PlayerScreen.tsx   → vote
✅ RoundDisplay.tsx   → tick (آخر 5 ثواني)
```

## 🎮 الاستخدام في الكود:

```typescript
// في المكونات
import { useSound } from '../hooks/useSound';

const { playSound } = useSound();

// تشغيل صوت
playSound('vote');           // حجم افتراضي 0.7
playSound('correct', 0.8);   // حجم 0.8
playSound('winner', 1.0);    // حجم 1.0 (كامل)
```

## ⚙️ التخصيص:

### تغيير الحجم:
```typescript
// في AdminScreen.tsx أو PlayerScreen.tsx
playSound('vote', 0.5);  // تغيير من 0.7 إلى 0.5
```

### إضافة أصوات جديدة:
```typescript
// في src/app/hooks/useSound.ts
export type SoundName =
  | 'vote'
  | 'spin'
  | 'tick'
  | 'correct'
  | 'wrong'
  | 'winner'
  | 'newSound';  // ← أضف هنا

export const SOUND_FILES: Record<SoundName, string> = {
  vote: '/sounds/vote.mp3',
  // ... باقي الأصوات
  newSound: '/sounds/new-sound.mp3'  // ← أضف هنا
};
```

## 🚨 استكشاف الأخطاء:

### الصوت لا يعمل؟

**1. تحقق من وجود الملفات:**
```bash
ls -lh public/sounds/
```

**2. تحقق من Console:**
افتح DevTools (F12) → Console
ابحث عن رسائل خطأ مثل:
```
Sound "vote" not found
Could not play sound "vote": NotAllowedError
```

**3. سياسة Autoplay:**
- بعض المتصفحات تمنع تشغيل الأصوات تلقائياً
- الصوت يعمل بعد أول تفاعل (ضغطة، نقرة)
- في الكود: يتم التعامل مع هذا تلقائياً (`catch` صامت)

**4. تنسيق الملفات:**
- استخدم MP3 (أفضل توافق)
- حجم الملف: 50-500 KB
- جودة: 128 kbps كافية

## ✨ نصائح:

### للأداء الأفضل:
- **أصوات قصيرة:** 0.5 - 2 ثانية
- **حجم صغير:** أقل من 100 KB لكل ملف
- **جودة متوسطة:** 128 kbps

### للتجربة الأفضل:
- **vote.mp3:** قصير جداً (< 0.5 ثانية)
- **spin.mp3:** متوسط (1-2 ثانية)
- **tick.mp3:** قصير جداً (< 0.3 ثانية)
- **correct.mp3:** متوسط (1-2 ثانية)
- **wrong.mp3:** قصير (0.5-1 ثانية)
- **winner.mp3:** طويل نسبياً (2-5 ثواني)
