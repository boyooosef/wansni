# 🎮 ونّسني - لعبة القعدة الكويتية

لعبة تفاعلية جماعية مبنية بـ React + Tailwind + Firebase

## 🌟 المميزات

- **نظام Admin & Players**: الأدمن يدير اللعبة واللاعبين يشاركون عبر موبايلاتهم
- **10 أقسام متنوعة**:
  - 🇰🇼 معلومات عن الكويت
  - 🧠 ألغاز
  - 🎭 أمثال كويتية
  - 🔥 تحديات
  - ⚡ سرعة بديهة
  - 🤔 تختار شنو؟
  - 😳 الصراحة
  - 🔥 تحديات أقوى
  - 🕌 أسئلة دينية
  - 🎬 الفن الكويتي

- **عجلة اختيار اللاعبين**: اختيار عشوائي مع رسوم متحركة
- **نظام التصويت**: اللاعبون يصوتون على نجاح أو فشل اللاعب
- **لوحة الصدارة المباشرة**: تحديث فوري للنقاط
- **QR Code للانضمام**: سهولة دخول اللاعبين عبر مسح الكود
- **Host TV Mode**: وضع العرض على الشاشة الكبيرة

## 📱 Progressive Web App (PWA)

اللعبة الآن تدعم PWA مع:
- ✅ **تثبيت على الموبايل**: يمكن تثبيت اللعبة كتطبيق على الجوال
- ✅ **تحديثات تلقائية**: يتم إشعار جميع المستخدمين تلقائياً عند توفر نسخة جديدة
- ✅ **عمل أوفلاين**: اللعبة تعمل حتى بدون إنترنت (ما عدا التزامن المباشر)
- ✅ **أيقونات مخصصة**: أيقونات اللعبة الخاصة (192x192 و 512x512)

### رابط اللعبة
```
https://your-domain.com/wansni
```

## 🚀 كيفية اللعب

### كأدمن:
1. افتح التطبيق واضغط "إنشاء قعدة كأدمن"
2. حدد نقاط الفوز (افتراضياً 70)
3. شارك رقم القعدة أو QR Code مع اللاعبين
4. اختر القسم وابدأ الجولات

### كلاعب:
1. افتح التطبيق وأدخل رقم القعدة
2. اكتب اسمك واختر الأفاتار
3. انضم للعبة وانتظر دورك
4. صوّت على أداء اللاعبين الآخرين

## 🛠️ التقنيات المستخدمة

- **Tailwind CSS v4** - التنسيق

## 🎯 نظام النقاط

- ✅ **النجاح**: يحصل اللاعب على نقاط السؤال (10-20 نقطة)
- ❌ **الفشل**: يخسر اللاعب 5 نقاط
- 🏆 **الفوز**: أول لاعب يصل لنقاط الفوز المحددة
